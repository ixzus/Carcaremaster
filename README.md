##Function module

### 技师和车主可以就车主发出的问题进行即时聊天（基于XMPP协议的openfire服务）.
### 针对公共问题，所有技师都可以看到，并进行抢单操作，先到先得.
### 技师可以对自己的可预约时间进行管理.
### 技师可以将自己在平台上的奖励进行提现操作，但是，这笔资金必须是进过平台审核无误并且过了一周之后才能被提现.
